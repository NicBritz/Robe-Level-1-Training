The second-generation Robe Navigation System (RNS 2)

This is the menu system used on the ROBIN® series.

XT series and those before used 7 segment displays and buttons to access menu items.

AT series introduced the Robe Navigation System (RNS), using a graphic display and encoder wheel to highlight and select menu items. This allowed easy access to a wider selection of parameters, and was groundbreaking in its time as even the market-leading brands were still using 7 segment displays.

The ROBIN® series utilizes touchscreen technology using an intelligent display, which provides a complete user interface to the feature range of these fixtures. With so many possibilities, the menu system on all ROBIN® fixtures is separated into logical sections called Tabs. The CycFX, MiniMe, Mini Pointe and LEDbeam100 use a small LCD display accessed by four buttons, but the category headings within the menu system are similar.
